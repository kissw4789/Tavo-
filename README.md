# Tavo 0.88 记忆与关系系统

这是第五季果汁插件包的 0.88 发布版，适配 TAV 0.88 的消息级 API。

当前推荐直接导入：

- `Tavo-0.88.json`

完整发布归档：

- `releases/v0.88/插件包6-18适配Tavo0.88.json`
- `docs/618发布结算文件.md`

## 这版解决什么

0.88 版把原来分散的记忆、回忆、关系能力收拢到统一插件包里，重点是让长篇角色扮演里的记忆更稳定、关系变化更可解释、旧用户数据有迁移入口。

这一版重点处理了三件事：

1. 记忆系统升级到 TAV 0.88 消息级 API。
2. 记忆世界书恢复自动绑定，关系世界书不自动绑定。
3. 关系系统阶段、输出规则、降级说明和多角色互动记录重新整理。

## 核心能力

### 记忆系统

- 自动整理短期总结、长期记忆和历史总览。
- 记忆世界书自动绑定，减少用户手动配置成本。
- 支持历史聊天补录、历史锚定楼层、从第一楼重新生成。
- 提供 TAV 0.88 升级提示：
  - 迁移旧数据：本地转换，通常不消耗 API。
  - 从第一楼重新生成：重新扫描历史聊天，会消耗 API。
- 待处理事项页新增筛选：
  - 待处理
  - 已完成
  - 全部
- 完成项不再因为更新时间被顶到待处理区前面。

### 回忆系统

- 保留重要瞬间、事件锚点和用户印记。
- 帮助入口统一为“使用说明”，已读后显示“帮助”。
- 打开首页、帮助和普通浏览不会自动消耗 API。

### 关系系统

- 统一六阶段：
  - 陌路
  - 初识
  - 往来
  - 纠葛
  - 羁绊
  - 宿命
- 阶段范围：
  - 陌路：0-4
  - 初识：5-14
  - 往来：15-39
  - 纠葛：40-64
  - 羁绊：65-89
  - 宿命：90-100
- 关系输出规则改为新版 `<RELATION>` 机器块。
- 支持 `interactions` 记录角色与角色之间的互动信号。
- 阶段不匹配时，系统会自动降级或挂起，并在界面里显示原因。
- 关系世界书核心规则使用更强的注入位置：
  - `atDepth`
  - 深度 `0`
  - 角色 `system`

## 导入方式

1. 下载 `Tavo-0.88.json`。
2. 在 Tavo 中导入插件包。
3. 打开总控入口。
4. 第一次使用记忆系统时，按弹窗选择：
   - 迁移旧数据
   - 从第一楼重新生成
   - 稍后处理
5. 需要旧聊天完整继承时，优先阅读弹窗说明后再操作。

## 注意事项

- 迁移旧数据通常不消耗 API，但旧格式复杂时仍建议先确认提示。
- 从第一楼重新生成会消耗 API。
- `interactions` 当前是角色间互动记录，不会直接改用户与角色的关系值。
- 关系系统仍属于内测能力，记忆系统是当前主要稳定使用场景。
